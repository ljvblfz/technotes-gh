#Google和各种网站的 IPv4 服务hosts
#该 hosts 文件修改自 Google和各种网站的IPv6服务hosts文件，原文链接https://docs.google.com/View?docid=dfkdmxnt_61d9ck9ffq
#本文件将上述文件提供的 IPv6 host 绝大多数更换为 IPv4 host 方便没有 IPv6 连接的童鞋使用
#如有侵权, 请到 G+ 上面发送信息, 我将予以删除 :P
#本 hosts 的发布地址：编辑中
#本 hosts 的共享地址：https://docs.google.com/document/d/1GWaVSxitymPsfUOXwl1BOD7rHR_RiolDoR13r1ydSMc
# 当前版本编辑于 Sep, 01, 2011 7:28 PM 
# 中国大陆内存在 DNS 污染现象, 建议将本机以太网连接的 IPv4 DNS 改为 Google 的 8.8.8.8 和 8.8.4.4
#用文本编辑器打开hosts文件，将以下内容复制进去，保存即可（hosts 文件没有后缀）
#Windows 用户可以以管理员身份直接运行 notepad "%SystemRoot%\system32\drivers\etc\hosts" 进行编辑
#Linux 用户在终端中执行 sudo gedit /etc/hosts 即可开始编辑
#Mac OS X 用户按住 Command+G 输入 /etc 即可打开含有 hosts 文件的窗口
#关闭某个IPv6的转发请在那一行的最前面添加#号，启用请去除最前面#号，每行中间的#号是为了区分地址和注释，不用理睬- -
#注意: 目前由于 IPv4 用户访问 YouTube 无法正常播放视频, 故本文中 YouTube 依然保持 IPv6 状态
#最后建议大家经常备份此文件, 以免由于被举报或 Google Docs 被和谐而无法获取最新的 hosts 文件
